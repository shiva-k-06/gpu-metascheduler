# Cognitive Self-Learning GPU Metascheduler for Heterogeneous HPC Clusters

A reinforcement learning-based GPU metascheduler that uses **Proximal Policy Optimization (PPO)** to intelligently schedule jobs across heterogeneous GPU clusters while optimizing scheduling performance, GPU utilization, workload balance, and thermal behavior.

---

## Project Overview

This project was developed as part of a research internship at **National Institute of Technology Tiruchirappalli (NITT)**.

The scheduler models GPU scheduling as a Reinforcement Learning problem where a PPO agent learns to assign incoming jobs to heterogeneous GPUs using multiple scheduling objectives.

The project also includes:

- Traditional scheduling algorithms for comparison
- Thermal-aware scheduling
- Reward function sensitivity analysis
- Dynamic cluster adaptability prototype

---

## Reward Function

The PPO agent learns using a multi-objective reward function consisting of

- Estimated Completion Time Penalty
- Queue Length Penalty
- Load Imbalance Penalty
- GPU Speed Bonus
- Memory Violation Penalty

A thermal penalty is additionally introduced for the thermal-aware scheduler.

---

## Dataset

No external dataset is used.

The workload is synthetically generated by `workload.py`, where every job receives randomized

- Arrival Time
- Runtime
- Memory Requirement
